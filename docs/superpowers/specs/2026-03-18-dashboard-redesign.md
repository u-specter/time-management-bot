# Dashboard Visual Redesign — Spec

**Date:** 2026-03-18
**Project:** Персональный тайм-менеджмент дашборд (Умиджон)
**Scope:** Визуальный редизайн существующего `index.html` — без изменения логики и данных.

---

## Решения дизайна

| Параметр | Выбор |
|---|---|
| Стиль | Glassmorphism (лёгкий) |
| Акцент | Янтарь / Золото (`#fbbf24`, `#f59e0b`) |
| Фон | `#0a0a0f` + 3 размытых ambient orb |
| Технологии | Bootstrap 5 + vanilla JS (один HTML файл) |

---

## Визуальная система

### Фон
- Цвет: `#0a0a0f`
- 3 ambient orb (position:fixed, blur:80px, pointer-events:none, z-index:0):
  - Правый верх: `rgba(251,191,36,0.07)`, 400×400px
  - Левый низ: `rgba(251,146,60,0.05)`, 300×300px
  - Центр: `rgba(99,102,241,0.04)`, 200×200px

### Glass-карточки (базовый компонент)
```css
background: rgba(255,255,255,0.03);
border: 1px solid rgba(255,255,255,0.07);
border-radius: 10px;
backdrop-filter: blur(8px);
```

### Янтарный акцент применяется к:
- Активный таб: `rgba(251,191,36,0.15)` bg + `rgba(251,191,36,0.25)` border + `color:#fbbf24`
- Активный день (day pill): аналогично
- Прогресс-бар: gradient `#f59e0b → #fbbf24` + `box-shadow: 0 0 8px rgba(251,191,36,0.4)`
- Выполненные галочки в целях
- ИТОГО в финансах
- Хедер — имя пользователя + дата

### Цветовые полоски расписания (cat-strip, без изменений)
gray `#6c757d` · blue `#0d6efd` · orange `#fd7e14` · green `#198754` · yellow `#ffc107` · purple `#6f42c1` · red `#dc3545`

## Компоненты по вкладкам

### Хедер (глобальный)
- Имя `Умиджон` + подпись `/тайм-менеджмент` — слева
- Текущая дата — справа, glass pill
- Отделён от контента

### Табы
- Единый glass-контейнер для всех 4 кнопок
- Активный таб — янтарный акцент
- Неактивные — `color:#555`

### Вкладка «День»
- 7 day-pill в ряд (flex), Вт/Чт/Сб — красный акцент, сегодня — янтарный
- Строки расписания: glass-карточки с `cat-strip` слева (3px, категорийный цвет)
- Выполненная строка: зачёркнутый текст, `rgba(251,191,36,0.04)` bg, янтарная галочка
- Прогресс-бар sticky внизу с blur backdrop

### Вкладка «Неделя»
- Glass-карточки для каждого дня
- Сегодня: янтарный border + badge
- Дни единоборств: красный левый border (2px)
- Задачи дня — теги-пилюли (`border-radius:4px`, `rgba(255,255,255,0.05)`)

### Вкладка «Финансы»
- Заголовок зарплаты с янтарной суммой
- Прогресс-бары расходов (height:6px, цвет = свой у каждой статьи)
- Glass-карточка итогов: ИТОГО выделено янтарным
- Трекер расходов: таблица с тёмными инпутами, подсветка отклонений (зелёный/красный)

### Вкладка «Цели»
- Glass-карточка на каждую из 9 областей (включая ♟ Шахматы 100→1200)
- Чекбоксы: янтарный фон при отметке
- Миниатюрный прогресс-бар снизу карточки (2px, янтарный)
- Общий прогресс внизу вкладки

## Дополнительные детали

### Ambient orbs (уточнение цветов)
- Правый верх: `rgba(251,191,36,0.07)` (янтарь)
- Левый низ: `rgba(251,146,60,0.05)` (оранжевый)
- Центр: `rgba(99,102,241,0.04)` (индиго)

### Glass-карточки (варианты)
- Основные карточки (расписание, цели, неделя, финансы): `opacity 0.03`, `blur(8px)`
- Sticky прогресс-бар: `rgba(10,10,15,0.92)`, `blur(16px)` — более тёмный

### Типографика
Без изменений — Bootstrap 5 defaults + system-ui stack.

### Hover / Focus состояния
- Строки расписания: `background: rgba(255,255,255,0.05)` on hover
- Day pills, tabы: лёгкое `opacity:0.8` или `rgba(255,255,255,0.05)` bg on hover
- Кнопки/pill — Bootstrap defaults, без кастомных focus rings

### Чекбоксы (цели)
Кастомный `<div class="goal-cb">` — при отметке: `background: rgba(251,191,36,0.2)`, `border-color: rgba(251,191,36,0.4)`, цвет галочки `#fbbf24`.

## Что НЕ меняется
- Вся JS-логика (localStorage, renderSchedule, initDayTab и т.д.)
- Все данные (SCHEDULE_NORMAL, SCHEDULE_MARTIAL, FINANCE_ITEMS, GOALS_DATA, WEEK_DATA)
- Структура HTML (те же id и классы, нужные JS)
- Одиночный HTML-файл без сборщиков
